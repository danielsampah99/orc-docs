# Application Process Payment Plan

On this page, users are presented with two available payment service options for processing their application:

- **Standard Plan**  
  Reliable submission processing with a standard turnaround time.

- **Premium Plan**  
  Enjoy faster submission processing with priority handling.

![Payment Plan Options](/img-professional-bodies/pb-img-6.png)

After reviewing the options, select your preferred payment plan and click the **Start Application** button to continue.
