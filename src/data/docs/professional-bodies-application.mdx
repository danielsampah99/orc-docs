# Start Application Page

The **Start Application** page introduces users to the concept of a **professional body**, providing context on its purpose and importance. This page includes an informative video to help users better understand what a professional body is and why membership may be beneficial.

In addition, the page outlines the **available payment plans** for processing a professional body application, allowing users to review their options before proceeding.

To begin the application process, users can simply click the **Start Application** button.
