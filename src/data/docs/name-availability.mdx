# Name Availability Check

On this page, users are required to enter the proposed name for registration in the provided text box, as shown below:

![Enter Name](/img-professional-bodies/pb-img-7.png)

Once the name is entered, click the **Check Availability** button to initiate a search.

- If the name is **not available**, a prompt will appear notifying you, as shown below:

![Name Not Available](/img-professional-bodies/pb-img-9.png)

- If the name **is available**, a confirmation message will be displayed, as shown below:

![Name Available](/img-professional-bodies/pb-img-8.png)

> **Disclaimer:**  
> Please note that name availability does not guarantee final approval. All names are subject to further review and compliance checks as outlined in the relevant regulations.

![Disclaimer Notice](/img-professional-bodies/pb-img-10.png)
